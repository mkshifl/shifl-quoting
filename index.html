Create a free Supabase account

Go to supabase.com → Sign up → Create a new project → wait ~2 minutes for it to provision.

2

Run the setup SQL

In your Supabase dashboard → SQL Editor → paste and run the SQL provided with this app (in the README or ask your developer).

3

Enter your connection details

Go to Supabase → Settings → API → copy your Project URL and anon public key below.

Your connection details are saved in this browser only — never shared with anyone.
